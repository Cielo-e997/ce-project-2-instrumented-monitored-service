# Runbook

## Checking System Health

1. Access the `/health` endpoint.
2. Verify that the application returns a healthy response.

---

## Inspecting Logs

1. Open CloudWatch Logs.
2. Navigate to the application log group.
3. Inspect recent log events.

---

## Checking Metrics

1. Open CloudWatch Metrics.
2. Review application metrics such as latency and request counts.

---

## Investigating Alerts

If an alarm is triggered:

1. Identify the metric that triggered the alarm.
2. Check the dashboard for abnormal patterns.
3. Inspect logs to identify the root cause.
