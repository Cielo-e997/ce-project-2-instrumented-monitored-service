# Monitoring

## Monitoring Dashboard

A CloudWatch dashboard was created to visualize important system metrics.

The dashboard provides a centralized view of system behavior and makes it easier to detect abnormal patterns.

Metrics displayed on the dashboard include:

- CPU utilization
- request latency
- request count
- error rates

---

## Golden Signals

The dashboard follows the concept of the Golden Signals used in site reliability engineering.

These signals help identify potential issues quickly.

The signals monitored in this project include:

- latency
- traffic
- errors
- saturation

Monitoring these signals allows abnormal system behavior to be detected early.

---

## Benefits of Monitoring

A monitoring dashboard allows system health to be assessed quickly.

Instead of inspecting individual metrics separately, the dashboard provides an overview that helps detect patterns and anomalies in system behavior.
